The Item Shelf

Description

The ItemShelf is a scriptableObject that holds data about all the items you set up, and the Inventory system unity package comes with a preconfigured ItemShelf containing a few items to help you get started.

The ItemShelf has to be located in a /Resources/ folder.
There can only be one ItemShelf per project.
